When try-catch block failed return else block otherwise try block.

Example:

var num = 0
tryCatchElse(elseBlock = { --num }) {
  throw Exception("throw")
  num++
} // -1

tryCatchElse(elseBlock = { --num }) {
  ++num
} // 0
